PART 1 : INTRODUCTION

PURPOSE OF THE PROJECT

Nowadays, there are universities and institutions are still manually adding their students’ data without using today’s technique. Universities’ staff must follow nowadays technologies and begin using applications that help them to do this daily work in such an effective way and save their time and data by using their own data base. Thus, the data will be saved without any loose or damage for long period.

OVERVIEW - INTRODUCTION

Zyad housam is pleased to submit this proposal for services to support Nisantasi in achieving its goals for it’s new tech vision by using its own unique platform that let Nisantasi’s admins to organize their data such as students emails, students numbers (id), students birthdate, and student’s city. All those can be used throw Nisantasi Organizer Application’s by easily accessing the data using database, for the UI (User interface) java with FXML using Scene Builder.
